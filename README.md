# Quanturion Pro – Premium Mobile Build

Diese ZIP ist für **Handy + GitHub + Expo** vorbereitet und berücksichtigt:
- Deutsch
- English
- עברית

## Was enthalten ist
- Premium Executive Savings Cockpit
- Steuer / Hypothek / Strom / Versicherung
- Premium-Bereich
- Rechtliches
- Sprachumschaltung DE / EN / HE
- Expo-Assets für Icon, Adaptive Icon und Splash

## Upload nur mit dem Handy

1. ZIP herunterladen
2. ZIP entpacken
3. In GitHub dein Repo `Quanturion-pro` öffnen
4. Alle Dateien aus dem Ordner `Quanturion-pro-main` hochladen bzw. ersetzen
5. Wichtig: **nicht** den ganzen Ordner als Extra-Unterordner hochladen

## So muss es in GitHub aussehen

- `App.js`
- `app.json`
- `eas.json`
- `index.js`
- `package.json`
- `README.md`
- `assets/icon.png`
- `assets/adaptive-icon.png`
- `assets/splash.png`

## Build ohne PC

Wenn GitHub schon mit Expo verbunden ist:

1. Expo Dashboard öffnen
2. Projekt `Quanturion Pro` auswählen
3. Android Build starten
4. APK wählen

## Hinweis
Der Code ist absichtlich ohne zusätzliche UI-Bibliotheken gebaut, damit er auf Expo stabiler bleibt.
